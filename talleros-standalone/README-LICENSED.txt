TallerOS Licensed v2.0
Instalador Completo con Electron
========================================

CONTENIDO
========================================
Este paquete incluye:
- TallerOS v2.0 completo (Backend + Frontend)
- Electron (aplicacion de escritorio)
- Sistema de activacion de licencias
- Base de datos SQLite (sin configuracion)

REQUISITOS
========================================
- Windows 10/11 (64 bits)
- 2 GB de espacio libre
- Conexion a internet (primera instalacion)

INSTALACION RAPIDA
========================================
1. Descomprime este ZIP en cualquier carpeta

2. Haz clic derecho en "Instalar-TallerOS-Licensed.bat"
   -> "Ejecutar como administrador"

3. Espera a que termine (5-10 minutos)
   - Instalara Node.js (si falta)
   - Instalara todas las dependencias
   - Creara accesos directos

4. Para iniciar:
   - Doble clic en el icono del escritorio "TallerOS Licensed"
   - O busca "TallerOS Licensed" en el menu inicio

ACTIVACION DE LICENCIA
========================================
La primera vez que inicies TallerOS:
1. Se abrira la ventana de activacion
2. Introduce tu clave de licencia
3. Sigue las instrucciones en pantalla

Para obtener una licencia:
- Web: https://igsm.es/talleros
- Email: licencias@igsm.es
